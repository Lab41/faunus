Data science is the exploration of phenomena via the use of empirical information (i.e. data). Data is gathered when measurements are taken of some world irrespective of that world being natural or synthetic. However, a graph typically represents some real-world system (except for in pure mathematics). Examples include social, biological, or technical systems. The study/analyses of graphs that model real-world systems is called network science. Faunus is a tool that can be utilized for network science on massive-scale graphs — when the graphs are so large that they no longer fit within the resource confines of a single machine. Moreover, Faunus is seen as a particular tool within the data science tool chain. A typical tool chain is enumerated below:




  1. World: a system that is intended to be understood via measurement and analysis.
  2. Graph: a graph representation of the worldly system in terms of “things” and their “connections.”
  3. Hadoop: a distributed storage and processing environment for analyzing massive-scale data sets.
  4. Faunus: a Hadoop-based framework for graph rewriting and/or yielding basic graph statistics.
  5. Statistics: mathematical tools for describing patterns in data in as few numbers as possible.
